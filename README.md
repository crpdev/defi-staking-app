# defi-staking-app

An imaginary "Decentral Bank", where users [accounts] can deposit a custom token [like USDT] and earn reward tokens [RWD].

Languages: Solidity, React JS
Platforms: Truffle, Ganache, Metamask
Tests: Mocha and Chai
